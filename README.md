## 설명
Servlet과 템플릿 엔진인 JSP를 이용하여, 여행 관련 게시판 구축.
## 기술 스택
HTML ,CSS, Javascript, Java,  Servlet, JSP, MySQL

## 처리된 요구사항 목록
